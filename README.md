# Vision Vortex: AI-Powered Learning for the Visually Impaired 👁️🌪️

**Vision Vortex** is a voice-first, multimodal educational ecosystem designed to provide **100% independence** for visually impaired students. Instead of simply reading text, Vision Vortex re-engineers the learning process through an adaptive AI loop that adjusts to a student's cognitive speed and reasoning ability.

---

## 🚀 The Innovation
Most accessibility tools are "Screen Readers." **Vision Vortex is a "Self-Correcting AI Tutor."** It eliminates the need for sighted assistance from login to exam prep by using advanced Spatial Reasoning and Cognitive Syncing.

## 🛠️ Key Features

### 1. Zero-Sighted Independent Login
Ensures the user never needs to ask for help to start.
* **Symbol/Webcam Mode:** Uses **Gemini 2.0 Flash** to recognize a specific object or hand gesture as a "Visual Key."
* **Voice Password:** Identity verification through unique voice biometrics.
* **Tactile Pattern:** Draw shapes on-screen guided by **Haptic Feedback** (vibrations).

### 2. Path Selection & Contextual IQ-Sync
* **Dual Environments:** Specialized modes for **Academic (K-12)** and **Competitive Exams (UPSC, IBPS, SSC)**.
* **Cognitive Assessment:** A 5-minute "IQ-Sync" evaluates logical reasoning and processing speed to set the initial **Learning Tier**.

### 3. Tier-Based Personalized Learning
The AI re-writes lessons in real-time based on the user's tier:
* **Foundation Tier:** Slower speech, high-clarity explanations, and simple analogies.
* **Advanced Tier:** Rapid information delivery and complex logical synthesis.

### 4. Multimodal Visual-to-Logic Decoding
Transforms visual barriers (Bar Graphs, Biology Diagrams) into **Logical Audio Narratives**.
* **Interaction:** Students can ask, *"Which year had the lowest sales?"* or *"What is the label on the X-axis?"* and the AI performs spatial reasoning to answer.

### 5. The Auditory "Mental Scratchpad"
Solve complex math without physical paper using the **Variable Vault**.
* Students verbally store numbers: *"Store interest rate as 8%."*
* The AI performs calculations using these stored variables later in the problem.

### 6. The Validation Loop (Topic-Wise Testing)
A strict **Learn → Test → Adapt** cycle:
* **Adaptive Feedback:** If a student struggles with a quiz, the AI automatically **downgrades complexity** and re-explains the concept using different analogies until mastery is achieved.
